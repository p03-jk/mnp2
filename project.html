Final Project

Smart Home Management with Line Bot Integration using ESP32
ระบบควบคุมอุปกรณ์ไฟฟ้าภายในบ้านแบบครบวงจรผ่าน Line Bot โดยใช้บอร์ด ESP32

วัตถุประสงค์ของโครงงาน

  • เพื่อศึกษาการเขียนโปรแกรมควบคุมอุปกรณ์ต่าง ๆ โดยใช้บอร์ด ESP32
  • เพื่อออกแบบพัฒนาระบบควบคุมอุปกรณ์ไฟฟ้าภายในบ้านแบบครบวงจร
  • เพื่อศึกษาการเขียนโปรแกรมสร้าง Line Bot และการใช้ API ต่างๆ

ประโชน์ที่ได้จากโครงงาน

  • ผู้ใช้สามารถควบคุมอุปกรณ์ไฟฟ้าภายในบ้านจากที่ไหนก็ได้ผ่าน LINE Bot
  • สามารถลดความเสี่ยงจากการใช้งานผิดพลาดภายในบ้าน
  • ได้ระบบควบคุมอุปกรณ์ไฟฟ้าภายในบ้านแบบครบวงจร
